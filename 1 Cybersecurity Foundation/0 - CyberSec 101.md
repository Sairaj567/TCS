## Course Overview

**Course Name:** Cybersecurity 101

This course provides a strong foundation in cybersecurity by introducing the core principles, threats, security technologies, best practices, and career pathways in the field.

### Course Objectives

By the end of this course, I will be able to:

- Understand the fundamentals of cybersecurity
- Learn common cyber threats and attacks
- Understand networking and endpoint security
- Learn encryption and data protection techniques
- Explore cybersecurity tools and best practices
- Understand social engineering attacks
- Learn legal and ethical aspects of cybersecurity
- Perform basic hands-on security exercises
- Build a cybersecurity mindset
